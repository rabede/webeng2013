Kapitel 6: Wichtige Aspekte zu HTML/CSS

Hausaufgabe: Mit CSS formatierte Tabelle aller "XHTML 1.0 strict"-Elemente

In dieser Hausaufgabe sollen Sie auf Ihrer Website bei cloudControl eine XHTML-1.0-strict-Seite erstellen (ohne Bootstrap!). Die Seite soll eine "schön" formatierte Tabelle aller 77 Elemente zeigen mit den folgenden Spalten:

  1. Element-Tag mit den spitzen Klammern: <tag/></tag> oder falls zulässig <tag />
  2. Elementbeschreibung Inline- oder Blockelement?
  3. Semantisches HTML?
  4. CSS und HTML der Seite sind mit dem W3C-Validator zu validieren (keine Fehler, keine Warnungen).
