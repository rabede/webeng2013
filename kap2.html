Kapitel 2: Starter Kit: Tools für die Website in der Cloud

Einführung: Die erste Herausforderung

Langsam wirds ernst, die benötigten Werkzeuge werden vorgestellt. Niemand muß sie alle von Hand installieren, das Kursteam hat eine virtuelle Maschine vorbereitet, die über Bittorrent geladen werden kann und den bequemen Direkteinstieg ermöglicht.

Zum Einsatz sollen kommen:

  • Virtualisierungssoftware VirtualBox
  • Apache HTTP-Server
  • Webbrowser Mozilla Firefox mit den Add-ons:
    • Firebug
    • YSlow
    • RESTClient
    • Web Developer
  • Integrierte Entwicklungsumgebung Eclipse
  • Website-Template Twitter Bootstrap
  • Versionsverwaltungssoftware Git
  • Cloud der Firma cloudControl

Einige alte Bekannte, einige interessant klingende neue Geschichten. Auf jeden Fall bin ich erleichtert, daß der Kurs mit Linux und Opensource-Software zu bestreiten ist. Die VM habe ich testweise installiert, aber auf meinem alten Laptop läuft sie nicht so schnell, daß man Spaß am Arbeiten haben könnte. Deshalb habe ich mir alle benötigten Werkzeuge doch direkt installiert, ist ja zum Glück unter Linux kein Problem.
